## Setup

Install the following:

- nodejs
- yarn
- live-server

Run the following:

```bash
yarn install
```

followed by

```bash
npm run build
```

followed by:

```bash
live-server public
```

Open `http://127.0.0.1:8080/`

Any changes to the CSS, re-run `npm run build` locally.
